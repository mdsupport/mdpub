Addons for OpenEMR (Public)
===========================

**mdpub** is a collection of Add-On modules for OpenEMR installations.
It can be installed using composer by itself or as a dependency for your private repo.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
